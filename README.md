# 🎯 Discord Username Sniper

A lightweight tool that checks the availability of **Discord usernames** and sends a notification to a **Discord webhook** when an available username is found.

This tool is designed for fast checking with automatic delay handling to avoid rate limits.

---

## 🚀 How To Use

1️⃣ Run **Sniper.exe**
2️⃣ Enter your **Discord Token**
3️⃣ Enter your **Webhook URL**
4️⃣ Choose the **username length**
5️⃣ Choose how many usernames to check

The program will automatically start checking usernames.

If an available username is found, it will be sent to your **Discord webhook**.
